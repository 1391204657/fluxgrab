Import cookies.txt (Bilibili & login sites)

Updated July 2026

Some sites (especially Bilibili) block automated downloads until you prove you are logged in. FluxGrab reads a standard cookies.txt file placed next to the desktop app.

Steps (Windows desktop)

  1. Open Chrome or Edge and log in to the site (e.g. bilibili.com).
  2. Install a trusted export extension such as Get cookies.txt LOCALLY from the Chrome Web Store.
  3. While on the site, export cookies as cookies.txt (Netscape format).
  4. Move cookies.txt into the same folder as FluxGrab.exe (not inside subfolders).
  5. Restart FluxGrab and paste the video link again.

Tips

  • Cookies expire — if downloads fail again, export a fresh file.
  • Only use cookies from accounts you own. Do not share your cookies.txt.
  • The web version cannot use cookies.txt — use the Windows desktop app for Bilibili.
